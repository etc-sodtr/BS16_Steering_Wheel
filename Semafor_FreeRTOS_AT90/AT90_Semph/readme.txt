Acest program ruleaza mai multe task-uri. 
-un task afiseaza pe LCD odata la 500ms: numarul de task-uri si daca butonul este apasat sau nu.
-un task care aprinde si stinge un led cu perioada de 1s
-un task care aprinde si stinge un led cu perioada de 1,5s
-un semafor pentru task-ul butonului. Cand acesta este apasat va taskul pentru buton va fi servit imediat.
Fiecare task este independent de celalalt.
Scheduler-ul va activa task-urile in functie de prioritatile acestora.
